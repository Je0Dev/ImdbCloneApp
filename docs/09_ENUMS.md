# Enumerations (`enums/`)

The enums folder contains enumeration types used throughout the application.

## Folder Structure

```
enums/
├── Genre.java
├── Ethnicity.java
└── ContentType.java
```

## Genre

Movie/Series genres:
- ACTION, ADVENTURE, ANIMATION, BIOGRAPHY, COMEDY, CRIME
- DOCUMENTARY, DRAMA, FAMILY, FANTASY, HISTORY, HORROR
- MUSIC, MYSTERY, ROMANCE, SCI_FI, SPORT, THRILLER
- WAR, WESTERN

Used for filtering and categorizing content.

## Ethnicity

Ethnicity categories for celebrities:
- UNKNOWN (default), CAUCASIAN, AFRICAN_AMERICAN, HISPANIC
- ASIAN, NATIVE_AMERICAN, OTHER

## ContentType

Type of content:
- MOVIE
- SERIES

Used to distinguish between movies and TV series.
